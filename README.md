# FrankPEPstein

Interactive Peptide Fragment Design pipeline.

## Usage

You can run the full pipeline interactively in Google Colab:

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/Joacaldog/FrankPEPstein/blob/main/FrankPEPstein.ipynb)

### Requirements

-   **Modeller License Key**: You will need to provide a valid Modeller academic license key when running the notebook.
-   **Receptor PDB**: Have your target protein structure ready (PDB format).

## Repository Structure

-   `FrankPEPstein.ipynb`: The main notebook.
-   `DB/`: Contains the minipockets database and dictionary files.
-   `utilities/`: Contains external tools (ADFR, Click) and config templates.
-   `scripts/`: Python scripts for the pipeline steps.
